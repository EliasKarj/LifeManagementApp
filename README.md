## Ominaisuudet

-  **Muistiinpanot:** Tekstimuistiinpanojen luonti, muokkaus ja poisto.
-  **Tietojen pysyvyys:** Datan tallennus laitteen paikalliseen muistiin (JSON).
-  **Käyttöliittymä:** Selkeä listanäkymä ja erillinen muokkaussivu.
-  **Tehtävälista (To-Do):** Yksinkertainen checklist hoidettavista asioista.
-  **Vedenjuonnin seuranta:** Visuaalinen loki päivittäiselle nesteytykselle.
-  **Budjettiseuranta Lite:** Nopea kulujen kirjaus (Summa + Kategoria).
-  **Päätöksenteko-apuri:** "Onnenpyörä" joka arpoo vaihtoehdon puolestasi.
-  **Dark Mode:** Tumma teema silmien säästämiseksi.
